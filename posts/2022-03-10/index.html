error while building reactui2 after bumping up docker version

I bumped up npm packages versions in package.json. There were errors. After updating package.json, I haven’t run npm install, but immediately went on to building docker image. But docker uses package-lock.json which hasn’t been updated. npm install seemed to have resolved the problem

Another error. Attempted import error: ‘Switch’ is not exported from ‘react-router-dom’ (imported as ‘Switch’). There were changes in react-router-dom package. Tests (npm t) haven’t passed. After changes from here they went ok.

Untill another error - setImmediate is not defined There are some hints here importing setImmediate from “timers” module solved the problem. Tests passed.
